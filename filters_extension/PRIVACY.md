# Privacy Policy — HubSpot Filter Copier

**Last updated: May 2026**

HubSpot Filter Copier is a Chrome extension that reads the active filters on HubSpot list views and copies a shareable URL to your clipboard.

## Data collection

This extension does not collect, store, transmit, or share any personal data or usage information. Specifically:

- No data is sent to any external server
- No analytics or tracking is performed
- No HubSpot data (contacts, companies, deals, or filter configurations) leaves your browser
- The only network requests made are to HubSpot's own API endpoints, using your existing HubSpot session, to read the current view's filter state

## Permissions

The extension requests the following Chrome permissions:

- **activeTab** — to read the URL of the current HubSpot tab
- **scripting** — to inject the filter-reading script into HubSpot pages
- **clipboardWrite** — to copy the generated URL to your clipboard
- **Host permission for hubspot.com** — to interact with HubSpot pages and their API

## Contact

This extension is open source. For questions or concerns, open an issue on the GitHub repository.
